# personnel-management-system-
基于 Spring Boot + Vue 的人事管理系统，是一款高效的企业管理工具。系统可实现员工信息管理、考勤统计、薪资核算、绩效考核等功能。Vue 构建的前端界面，操作简单直观，便于员工和管理人员使用；后端依靠 Spring Boot 架构，保障系统稳定，数据存储安全。该系统可有效提升人事工作效率，实现企业人事管理的信息化、规范化。
